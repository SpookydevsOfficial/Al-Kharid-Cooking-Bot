## Installation & Running the Bot

Follow these steps to add and run the bot on your local OSBot setup:

1. **Download the Bot Script**
   - Save the bot `.jar` file (or project folder if building from source) to your computer.

2. **Add the Script**
   - If you downloaded a `.jar` file, move it directly into the `OSBot/Scripts` folder inside the directory.
   - If you have the source code:
     - Open the project in an IDE (like IntelliJ IDEA or Eclipse).
     - Build the project into a `.jar` file.
     - Move the compiled `.jar` to the `OSBot/Scripts` folder.

3. **Run the Script**
   - Go back to OSBot.
   - Click `Scripts` & Refresh the List
   - Find the bot in the list and click `Start` in or around the bank/range.
   - Watch as it automates cooking in Al-Kharid!
